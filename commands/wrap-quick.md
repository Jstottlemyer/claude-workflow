This is `/wrap` with the `quick` argument. Read `commands/wrap.md` in this repo and execute it now as if `$ARGUMENTS` contained `quick` — the rest of the workflow is defined there.

Quick mode skips Phase 1b (insights), Phase 2c (wiki), Phase 3 (loose ends), Phase 3b (deps), and Phase 5 (CLAUDE.md health). Use when you just want the session summary + learning triage and nothing else — typically a 30-second wrap.

This file exists as a tab-completion shortcut. Behavior is identical to typing `/wrap quick`.
